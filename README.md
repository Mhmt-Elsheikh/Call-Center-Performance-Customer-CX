Call Center Performance & Customer CX

Description: Analyzed 5,000+ support calls to optimize customer service efficiency. The dashboard tracks critical KPIs like Average Speed of Answer (67.53s) and Customer Satisfaction (3.40), while providing granular insights into agent productivity and call resolution rates. It helps management identify bottlenecks in specific topics like "Payment" or "Tech Support."

Tech Stack: Power BI, DAX, KPI Monitoring.
